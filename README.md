
# 🏡 Be My Stay

Sistema web de reservas de imóveis inspirado em plataformas como Airbnb. O projeto permite que usuários realizem reservas, enquanto administradores podem controlar usuários, imóveis e gerar relatórios.

---

# 📌 Descrição

O **Be My Stay** é uma aplicação web desenvolvida com foco na gestão de hospedagens e reservas. O sistema permite que usuários encontrem imóveis disponíveis em diferentes cidades, visualizem detalhes das propriedades e realizem reservas com datas de check-in e check-out.

Além disso, o sistema possui funcionalidades administrativas para gerenciamento de usuários, imóveis e geração de relatórios.

---

# 🚀 Funcionalidades

## 👤 Usuários
- Cadastro e login de usuários
- Busca de imóveis por cidade
- Visualização de detalhes do imóvel
- Realização de reservas
- Visualização de reservas passadas e futuras

## 🏠 Imóveis
- Cadastro de imóveis
- Edição e exclusão de imóveis
- Associação de comodidades
- Upload de imagens
- Controle de disponibilidade baseado em datas de reserva

## 📅 Reservas
- Criação de reservas com **check-in e check-out**
- Verificação automática de conflito de datas
- Listagem de reservas do usuário

## 🛠 Administração
- Gerenciamento de usuários
- Gerenciamento de imóveis
- Gerenciamento de comodidades
- Gerenciamento de métodos de pagamento
- Gerenciamento de tipos de imóveis e lugares
- Dashboard com contadores do sistema
- Relatórios em PDF

## 📊 Relatórios
- Usuários com mais reservas
- Imóveis mais lucrativos
- Imóveis cadastrados no último mês
- Total de reservas ativas
- Exportação em **PDF**
